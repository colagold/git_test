## 分支管理

### 创建分支
```git
git branch 分支名
```

### 切换分支
```git
git checkout 分支名
```

或者
```git
git switch 分支名
```

### 创建并切换分支
```git
git checkout -b 分支名
```

或者
git switch -c 分支名


### 合并分支
```git
git merge 分支名   # 将指定分支合并到当前分支
```

### 删除分支
```git
git branch -d 分支名       # 已合并分支
git branch -D 分支名       # 强制删除
```
